Instruction Manual: Entering Stock Updates

Note: This manual provides step-by-step instructions for updating inventory levels when stock is added or removed.

Overview

Keeping accurate inventory records requires updating stock levels whenever materials are added to or removed from inventory. This process allows you to track inventory changes, maintain accurate stock counts, and monitor usage patterns.

Methods for Entering Stock Updates

There are two ways to enter stock updates:

  1. From the Stock Update Entry page (covered in this manual)
  2. Directly from the Consumable Materials list (covered in the third manual)

Step 1: Access the Stock Update Entry Page

From the home page, click on the "Stock Updates" card, then click the "Add New Entry" button at the top of the page.

Step 2: Select an Item

From the dropdown menu, select the consumable material you want to update. Once selected, the system will automatically populate the item details including:

  • Item Short Code
  • Normal Location
  • Current Quantity
  • Reorder Threshold

Step 3: Choose the Action Type

Select whether you are:

  • Adding Items: Increasing inventory (e.g., when new stock arrives)
  • Removing Items: Decreasing inventory (e.g., when items are used or taken)

The default action is "Remove Items".

Step 4: Enter the Quantity

Specify how many units are being added or removed. The minimum quantity is 1.

Note: The system will automatically calculate the new total quantity based on your entry.

Step 5: Select the Employee

From the dropdown menu, select the employee who is adding or removing the items. This helps track who made each inventory change.

Step 6: Add Notes (Optional)

You can add notes about this inventory change, such as the reason for removal or the source of new items.

Step 7: Submit the Update

Review all information for accuracy, then click the "Submit Entry" button. If successful, you will be redirected to the Stock Updates list where your new entry should appear at the top.

Important: Once a stock update is submitted, the inventory quantity is immediately updated in the system. If you make a mistake, you'll need to create a new entry to correct it.

Understanding the Stock Updates List

After submitting an update, you'll see it in the Stock Updates list. This list shows:

  • ID: The unique identifier for the update
  • Short Code: The item's short code
  • Name: The item's full name
  • Location: Where the item is stored
  • Added/Removed: How many units were added or removed
  • New Quantity: The updated total quantity
  • Employee: Who made the change
  • Date: When the change was made
  • Actions: Options to edit the entry
Tip: If an item falls below its reorder threshold after a removal, it will be highlighted in the Consumable Materials list.

Editing a Stock Update

If you need to edit a recent stock update:

  1. Find the entry in the Stock Updates list
  2. Click the "Edit" button in the Actions column
  3. Make your changes in the form that appears
  4. Click "Submit Entry" to save your changes
Note: Editing a stock update will recalculate the inventory quantity based on your changes.
