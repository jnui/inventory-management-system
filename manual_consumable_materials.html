Instruction Manual: Using the Consumable Materials Page

Note: This manual provides step-by-step instructions for using the Consumable Materials page to find items and make quick stock changes.

Overview

The Consumable Materials page is the central hub for viewing and managing your inventory. It allows you to:

  • View all materials in the inventory
  • Filter and search for specific items
  • Make quick stock updates directly from the list
  • Identify items that need reordering

Part 1: Navigating the Consumable Materials Page

Step 1: Access the Consumable Materials Page

From the home page, click on the "Consumable Materials" card to access the materials list.

Step 2: Understand the Page Layout

The page contains several key elements:

  • Action Buttons: At the top of the page (Add Material, Show/Hide Columns, Print, Excel)
  • Search Box: For quickly finding items
  • Materials Table: Displaying all inventory items with their details
  • Action Buttons: In the rightmost column of each row (Edit, Stock Chg, Delete)

Part 2: Filtering and Finding Items

Step 1: Use the Search Box

The search box at the top of the page allows you to quickly filter items. It searches across all visible columns.

  • Type any keyword, item name, code, or location
  • Results will update instantly as you type
  • The search is case-insensitive and works with partial matches

Step 2: Show/Hide Columns for Better Filtering

You can customize which columns are visible to focus on specific information:

  1. Click the "Show/Hide Columns" button at the top of the page
  2. Check or uncheck columns in the dropdown menu
  3. The table will instantly update to show only the selected columns

Step 3: Sort Items by Column

You can sort the table by any column:

  1. Click on any column header to sort by that column
  2. Click again to reverse the sort order (ascending/descending)
  3. A small arrow will indicate the current sort direction
Tip: Items that need reordering (below threshold) are highlighted in yellow for easy identification.

Part 3: Making Stock Changes from the Materials List

Step 1: Find the Item to Update

Use the search and filtering methods described above to locate the specific item you want to update.

Step 2: Click the "Stock Chg" Button

In the Actions column (rightmost column) of the item's row, click the "Stock Chg" button. This will take you directly to the Inventory Entry page with the item pre-selected.

Step 3: Complete the Stock Update Form

The Inventory Entry form will open with the selected item already filled in. Follow these steps to complete the update:

  1. Verify that the correct item is selected
  2. Choose whether you're adding or removing items
  3. Enter the quantity being added or removed
  4. Select the employee making the change
  5. Add any notes if needed
  6. Click "Submit Entry" to save the update

Step 4: Verify the Update

After submitting, you'll be redirected to the Stock Updates list where you can verify your entry was recorded. To return to the Consumable Materials list, click the home icon in the navigation bar, then click the "Consumable Materials" card.

Additional Features

Exporting Data

You can export the materials list for reporting or analysis:

  • Print: Click the Print button to print the current view of the table
  • Excel: Click the Excel button to download the data as a CSV file that can be opened in Excel
Note: Only the currently visible and filtered rows will be included in exports.

Editing Material Details

To edit an item's details (not its quantity):

  1. Find the item in the list
  2. Click the "Edit" button in the Actions column
  3. Make your changes in the form that appears
  4. Click "Submit" to save your changes
Important: The "Edit" button changes the item's details but not its quantity. To change quantity, use the "Stock Chg" button.
